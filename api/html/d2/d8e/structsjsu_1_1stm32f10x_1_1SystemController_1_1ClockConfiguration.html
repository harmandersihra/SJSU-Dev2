SystemClockSelect sjsu::stm32f10x::SystemController::ClockConfiguration::system_clock = SystemClockSelect::kHighSpeedInternal

Defines which clock source will be use for the system.

Warning
System will lock up in the following situations:
  • Select PLL, but PLL is not enabled
  • Select PLL, but PLL frequency is too high
  • Select High Speed External, but the frequency is kept at 0_Mhz.
