Detailed Description

The GetInactive() returns an implementation of an L1 peripheral that does nothing. We call these peripherals the inactives. They are meant to do nothing, or return trivial values when created and called. This is meant in cases where a particular module requires a particular peripheral to operate, but you want to substitute it out with a dummy implementation.
