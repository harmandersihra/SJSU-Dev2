Detailed Description

Clock configuration structure for use the lpc40xx microcontrollers See page 21 of a diagram of the clock tree that this datastructure represents: datasheets/sjtwo/LPC40xx/UM10562.pdf

See also

◆ multiply

uint8_t sjsu::lpc40xx::SystemController::ClockConfiguration::multiply = 1

The output rate multiplication factor of the PLL. If the input is 12 MHz and the multiply constant is 2, then the output frequency will be 24 MHz. The limit for this can be found in the datasheets/sjtwo/LPC40xx/UM10562.pdf or datasheets/lpc40xx/UM10562.pdf
