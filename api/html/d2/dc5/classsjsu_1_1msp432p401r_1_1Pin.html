reinterpret_cast<DIO_PORT_Interruptable_Type *>(PJ)

Array containing the port structures that map P1 to P10. Each port structure contains an odd and even port. For example, PA contains P1 and P2, PB contains P3 and P4, and so on. The only exception is Port J which is its own port and only has pins 1-5.
