Detailed Description

A controller for small speakers and piezoelectric buzzer.

Usage:

sjsu::Pwm & buzzer_pwm = GetBuzzerPwm();
sjsu::Buzzer buzzer(buzzer_pwm);
buzzer.Initializer();
// Set buzzer to 550 Hz @ 75% volume.
buzzer.Beep(550_Hz, 0.75f);
