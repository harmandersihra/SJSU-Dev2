Detailed Description

An abstract interface for timer peripherals. This interface can be used for keeping time, starting and stopping a hardware timer, and causing interrupts when the timer reaches a certain condition. This is not meant to be used for PWM generation, or other advanced feature.

(INCOMPLETE PERIPHERAL has not migrated to sjsu::Module)

Parameters
