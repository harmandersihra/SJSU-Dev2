Detailed Description

template<size_t kTaskStackSize>
class sjsu::rtos::PeriodicTask< kTaskStackSize >

An abstract layer for FreeRTOS tasks that are scheduled and executed at periodic intervals by the PeriodicScheduler.

Attention
This task inherits from the Task interface and must be persistent or in global space.
Template Parameters
kTaskStackSizeTask stack size in bytes.
