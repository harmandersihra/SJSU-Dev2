Oscillator sjsu::msp432p401r::SystemController::ClockConfiguration_t::clock_source = Oscillator::kReference

On reset, the master clock is driven by the digitally controlled clock.

Clock source for the backup clock.

On reset, the subsystem master clocks are driven by the DCO clock.

Note
Only the reference clock, very low frequency clock, or the low frequency clock can be used as the clock source.
The specified clock source is used to drive both the subsystem master clock and the low speed subsystem master clock.
Only the reference clock or the low frequency clock can be used as the clock source.
