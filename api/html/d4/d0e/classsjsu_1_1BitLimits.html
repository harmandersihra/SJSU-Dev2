Detailed Description

template<uint8_t bitwidth, typename IntType>
class sjsu::BitLimits< bitwidth, IntType >

BitLimits defines compile-time functions that provides Min() and Max() functions which calculate the minimum and maximum possible values for an integer of some number of bits that can fit within the integer type supplied.

Template Parameters
bitwidth- the number of bits represented in the value.
IntType- the container integer for the type.
