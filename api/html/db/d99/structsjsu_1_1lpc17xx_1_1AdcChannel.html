Detailed Description

Namespace containing predefined Channel_t description objects. These objects can be passed directly to the constructor of an lpc40xx::Adc object.

Usage:

sjsu::lpc17xx::Adc adc(sjsu::lpc17xx::AdcChannel::kChannel0);
