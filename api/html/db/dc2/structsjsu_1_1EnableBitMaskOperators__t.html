Detailed Description

template<typename>
struct sjsu::EnableBitMaskOperators_t< typename >

This struct is used in the template evaluation to determine if bitwise operators are enabled. This generic instance will match with all enumeration types that do not have their own template specializations. This will disable bit mask operations for all types. See the comments for SJ2_ENABLE_BITMASK_OPERATIONS for more details for the template specialization.

Template Parameters
Ifany other type that does not have a type specialization for EnableBitMaskOperators_t is used, the kEnable flag will be declared as false.
