constexpr size_t sjsu::infrared::DataFrame_t::kMaxPulseBufferSize = 67
inlinestatic

Maximum size of the pulse_buffer. A value of 67 is chosen since formats using pulse duration modulation consists of a header (2 pulses), up to 32 bits of data (64 pulses), and a stop pulse.
