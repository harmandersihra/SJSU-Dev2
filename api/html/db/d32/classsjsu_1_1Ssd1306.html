Detailed Description

Display driver for the SSD1306 OLED display driver chip, usually found on 0.98" oled displays. User manual: https://cdn-shop.adafruit.com/datasheets/SSD1306.pdf

reset- gpio connected to the reset line. If an RC reset circuit is used, this can be replaced with sjsu::GetInactive<sjsu::Gpio>()
clock_rate- clock rate to communication with the OLED display with.
