Parameters
canbus- canbus peripheral to use
device_id- Device ID of the RMD motor. This can be modified by the switches on the back of the motor. Default ID set by factory on the switches is 0x140.
params- RMD-X series parameters that must be set correctly for proper operation. If the gear ratio is incorrect, then the servo angles will be off.
