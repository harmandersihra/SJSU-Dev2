Detailed Description

template<typename... Params>
struct sjsu::LogInfo< Params >

Specialized log object that labels the log with a preceeding "INFO" label. Will only log if the SJ2_LOG_LEVEL is level SJ2_LOG_LEVEL_INFO or greater.

Params- Variadic type array to describe the params variable pack.
