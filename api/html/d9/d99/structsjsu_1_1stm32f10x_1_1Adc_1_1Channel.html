Detailed Description

Namespace containing predefined Channel_t description objects. These objects can be passed directly to the constructor of an stm32f10x::Adc object.

Usage:

sjsu::stm32f10x::Adc adc(sjsu::stm32f10x::Adc::Channel::kChannel0);
