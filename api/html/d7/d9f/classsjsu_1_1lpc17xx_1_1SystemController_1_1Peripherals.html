sjsu::lpc17xx::SystemController::Peripherals Class Reference

#include <system_controller.hpp>

Detailed Description

LPC17xx Peripheral Power On Values: The kDeviceId of each peripheral corresponds to the peripheral's power on bit position within the LPC17xx System Controller's PCONP register.

Note
The following bits are reserved by the manufacturer: 0, 5, 11, 20, 28.
