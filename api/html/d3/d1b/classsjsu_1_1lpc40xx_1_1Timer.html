Parameters
counter_frequency- the frequency that the timer's count register will increment by. If this is set to 1'000'000Hz then the counter will increment every microsecond. register will be 10 ms.
callback- a callback that will be called when the condition set by SetTimer method has occurred.
priority- sets the Timer interrupt's priority level, defaults to -1 which uses the platforms default priority.

Parameters
