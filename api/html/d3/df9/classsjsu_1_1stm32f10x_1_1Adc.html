Detailed Description

ADC driver for implementation for the stm32f10x microcontrollers. Activating a single channel will activate and enable sampling for all channel.

DMA channels used are:

  • DMA 1 Channel 1
  • DMA 2 Channel 5

Only supports ADC1 and ADC2. ADC3 not supported.
