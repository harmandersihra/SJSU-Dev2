◆ Flush()

virtual void sjsu::Uart::Flush ( )
inlinevirtual

Will flush all bytes currently head with the UART peripherals buffers.

The default implementation reads out all of the bytes by checking HasData() repeatedly and use Read() to read each. Some implementations may have more efficient methods of clearing their buffers.

Reimplemented in sjsu::stm32f10x::UartBase.
