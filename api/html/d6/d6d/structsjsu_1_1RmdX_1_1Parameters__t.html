◆ gear_ratio

float sjsu::RmdX::Parameters_t::gear_ratio

Reducer gear ratio. The RMD-X series has the encoder placed on the input motor and not the output motor, meaning that 360 rotation on the encoder is only 1/6 rotation of the output motor shaft if the motor's reducer is 6:1, thus this field is important to get proper angle mapping when using position control.
