◆ rx

sjsu::Pin& sjsu::stm32f10x::UartBase::Port_t::rx

Reference to the UART RX pin This driver will not handle remapping (muxing) the pin. So for example, For USART3 the default TX is PB10 and the default RX is PB11. But PC10 and PC11 can be remapped to USART3. In order to use those other pins, the AFIO->MAPR register must be configured for this before using this library, otherwise, the default mapping will be used.
