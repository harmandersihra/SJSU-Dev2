Detailed Description

An abstract interface for the Task interface class.

Note
The TaskInterface class is used to provide a generalization for the Task class since the Task class takes in a template argument. The Task interface class should be inherited instead of directly inheritting TaskInterface when creating a FreeRTOS task class.

virtual void sjsu::rtos::TaskInterface::SetDelayTime ( uint32_t  time)
pure virtual

Sets the delay time to ensure Run() is called at a desired frequency for periodic tasks.

For examples, if time = 1000 and the sjsu::rtos tick period = 1ms, then Run() will be called every 1 second.

Parameters
