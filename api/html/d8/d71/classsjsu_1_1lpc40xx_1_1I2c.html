void sjsu::lpc40xx::I2c::ConfigureClockRate ( units::frequency::hertz_t  frequency = 100 '000_Hz,
float  duty_cycle = 0.5 
)
inlineoverridevirtual

MUST be called before running Enable(). Set the clock rate and duty cycle of the serial clock frequency.

Parameters
frequency- the clock frequency to set the I2C serial clock frequency to.
duty_cycle- percentage of time the clock rate is LOW. Moving this value closer to 0 will cause the time the serial clock is low to be shorten. This will allow more time for the signal to pull high via the external pull up resistor.

Implements sjsu::I2c.
